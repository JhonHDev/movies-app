# Movies React Native App

# Step 1: Environment variables

Adding the required environment variables to the .env file

```
THE_MOVIE_DB_API_KEY=
```

## Step 2: Start the Metro Server

```bash
# using npm
npm start

# OR using Yarn
yarn start
```

## Step 3: Start your Application

```bash
# using npm
npm run android

# OR using Yarn
yarn android
```

### For iOS

```bash
# using npm
npm run ios

# OR using Yarn
yarn ios
```
